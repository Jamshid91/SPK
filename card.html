Описание

Борцовки для самбо ESKHATA изготовленные из сочетания натуральной кожи, велюра и полиэстера, предназначены для тренировок и участия в соревнованиях всех уровней по спортивному и боевому САМБО. Борцовки для самбо ESKHATA разработаны при взаимодействии с ведущими спортсменами и тренерами по Самбо. Борцовки обеспечивают легкость и удобство для тренировок. Подошва борцовок выполнена из натуральной кожи с внутренней стелькой из полиуретана. Борцовки произведены по колодке и держат форму.
